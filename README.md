# 基于springboot的鲜花管理系统16



## 1、项目介绍

基于springboot的鲜花管理系统16拥有两种角色

管理员：用户管理、分类管理、订单管理、发货、商品管理等

用户：登录注册、购买、购物车、订单、收货

![架构](https://www.codeshop.fun/Typora-Images/202402171137135.jpg)

## 2、项目技术

后端框架：springboot

前端技术：html

## 3、开发环境

- JAVA版本：JDK1.8（最佳）
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.7、8.x版本都可） 
- tomcat版本：无需
- maven版本：无限制
- 硬件环境：Windows
